Installation & Setup

Requirements

  • Device: Android with Termux installed.
  • Storage: Min 3GB free. (The Radio feature requires more storage; images and recordings also consume space.)
  • RAM: Min 2GB.

Step-by-Step Installation

  1. Install Termux & Add-Ons

    Note: To install APKs (e.g., F-Droid), ensure you:
    - Enable unknown sources (Settings > Security > Install Unknown Apps).
    - Download F-Droid, then get Termux from F-Droid.
    - Install add-ons: Termux:API, Termux:Styling.
    - Allow the fdroid process when prompted.

  2. Run Setup Commands
    Important: Open the Termux application on your device before copying and pasting the commands below.
    termux-setup-storage && pkg update -y && pkg upgrade -y && pkg install aapt clang cloudflared curl ffmpeg fzf git jq libffi libffi-dev libxml2 libxslt nano ncurses nodejs openssh openssl openssl-tool proot python rust unzip wget zip -y
    pip install blessed bs4 cryptography flask flask-socketio geopy mutagen phonenumbers pycountry pydub pycryptodome requests werkzeug
    pkg install termux-api
  3. Clone Repository
    git clone https://github.com/dedsec1121fk/DedSec

    Tip: If you get an error like "ModuleNotFoundError: No module named 'requests'", use this command and try again:

    pip install requests
  4. Choose Menu Style
    cd DedSec/Scripts && python Settings.py

    Tip: You can open the menu later by typing m in Termux.

Toolkit Summary

  1. Fox Chat
    Encrypted chat, along with file sharing, voice messages, and video calls.
  2. Android App Launcher
    Displays and manages downloaded Android apps.
  3. Radio
    Features an offline radio with Greek and international artists.
  4. Phishing Attacks
    Captures camera images, audio, and the target's exact location (address/store). It can also obtain credit card and personal information. All captured data is saved in the Downloads folder.
    • Fake Card Details Page.py: Saves credit card info to ~/storage/downloads/CardActivations/.
    • Fake Data Grabber Page.py: Saves personal info/photo to ~/storage/downloads/Peoples_Lives/{user}/.
    • Fake FaceFriends Page.py: Stores credentials to ~/storage/downloads/FaceFriends/.
    • Fake Google Free Money Page.py: Saves credentials to ~/storage/downloads/GoogleFreeMoney/.
    • Fake Google Location Page.py: Stores location data (GPS, address, IP) to ~/storage/downloads/Locations/.
    • Fake Insta Followers Page.py: Saves credentials to ~/storage/downloads/FreeFollowers/.
    • Fake Back Camera Page.py: Stored images/credentials to ~/storage/downloads/Camera-Phish-Back/.
    • Fake Front Camera Page.py: Saves front camera photos/credentials to ~/storage/downloads/Camera-Phish-Front/.
    • Fake Location Page.py: Stores location data (GPS, address, IP) to ~/storage/downloads/Locations/.
    • Fake Microphone Page.py: Saves mic recordings/credentials to ~/storage/downloads/Recordings/.
    • Fake Snap Friends Page.py: Stores credentials to ~/storage/downloads/SnapFriends/.
    • Fake Tik Followers Page.py: Saves credentials to ~/storage/downloads/TikFollowers/.
  5. Settings
    Allows you to update the project, install or update packages and modules, change your username and menu style, and view credits.
  6. DedSec Database
    Lets you upload, search for, and delete files. Your device acts as the server, and files are saved in a hidden folder.
  7. HTML Finder
    A utility to inspect, download, and locally edit any public webpage.
  8. URL Masker (Masker.py)
    A tool to conceal and shorten URLs.
  9. SocialBrute
    An advanced multi-service brute-force tool for social media platforms.
  10. Extra Content
    Contains photos, unused scripts, voice-overs, and more.

    Ανέβασμα, αναζήτηση και διαγραφή αρχείων σε τοποτικό διακομιστή.
