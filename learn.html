Learn About The Tools

Discover the capabilities and educational purpose of each tool in the DedSec Project

If you own or you're working with any of the official pages that are used in the fake pages and you want us to remove them, Contact us through the contact page and let us know!

Toolkit Summary

Android App Launcher

A powerful Android application manager that runs directly on Android devices through Termux.

How to Use:

Run the script in Termux to see all installed apps.

File Storage:

Extracted APKs are saved to: ~/storage/shared/Download/Extracted APK's/.

DedSec's Network

A comprehensive network and security toolkit optimized for Termux and low-resource devices.

How to Use:

Launch the script to access the main TUI menu.

File Storage:

All files are saved in: ~/DedSec's Network/ directory.

Digital Footprint Finder

An advanced OSINT tool for discovering digital footprints across multiple platforms.

How to Use:

Run the script and use the TUI interface to enter usernames.

File Storage:

Results are saved to: ~/storage/shared/Download/Digital Footprint Finder/.

File Converter

An advanced, interactive file converter for Termux using 'curses' with support for 40 different file formats.

How to Use:

Run the script to access the curses-based UI.

File Storage:

All conversion folders are organized in: /storage/emulated/0/Download/File Converter/.

Fox's Connections

A unified communication platform combining Fox Chat and DedSec's Database with a single login.

How to Use:

Run the script and create a one-time secret key.

File Storage:

Database files are stored in: ~/Downloads/DedSec's Database/ directory.

Loading Screen

A customizable ASCII art loading screen for Termux that displays during startup.

How to Use:

Run the script and choose from the menu options.

Configuration:

Modifies ~/.bash_profile and bash.bashrc configuration files.

Masker

A URL masking tool that creates shortened, human-readable aliases for URLs using multiple shortening services.

How to Use:

Run the script and paste the URL to mask.

Features:

Generates human-readable aliases with automatic service fallback.

Personal Information Capture

A comprehensive information gathering tool that can capture camera photos, microphone recordings, location data, and other device information.

How to Use:

Run the script and follow the menu options to select which information to capture.

File Storage:

All captured data is saved in folders inside your internal storage Downloads folder.

QR Code Generator

A simple yet powerful QR code generator that creates QR codes from URLs and saves them directly to the phone's Downloads folder.

How to Use:

Run the script and enter the URL you want to encode.

File Storage:

QR codes are saved to: ~/storage/downloads/QR Codes/ directory.

Settings

The central configuration hub for the DedSec Project, providing system management, updates, language selection, menu customization, and project information.

How to Use:

Run the script to access the main settings menu.

Configuration:

Settings are stored in bash.bashrc for startup configuration.

Smart Notes

An intelligent note-taking application with reminder functionality, external editor integration, and both TUI and CLI modes.

How to Use:

Run without arguments for TUI mode, or use --add for CLI note creation.

File Storage:

Notes are stored in: ~/.smart_notes.json.

Social Media Fake Pages

A collection of scripts that can create fake pages for famous social media platforms and other websites.

How to Use:

Run the specific script for the social media platform you want to demonstrate.

File Storage:

All generated pages and their assets are saved in folders inside your internal storage Downloads folder.
