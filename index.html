How To Install And Setup The DedSec Project

Requirements

  • Device: Android with Termux installed.
  • Storage: Min 8GB free.
  • RAM: Min 2GB.

Step-by-Step Installation

  1. Install Termux & Add-Ons

    Note: To install APKs (e.g., F-Droid), ensure you:
    - Enable unknown sources (Settings > Security > Install Unknown Apps).
    - Download F-Droid, then get Termux from F-Droid.
    - Install add-ons: Termux:API, Termux:Styling.
    - Allow the fdroid process when prompted.

  2. Update Packages & Install Git
    Important: Open the Termux application on your device before copying and pasting the command below.
    pkg update -y && pkg upgrade -y && pkg install git nano -y && termux-setup-storage
  3. Clone the Repository
    git clone https://github.com/dedsec1121fk/DedSec
  4. Run the Setup Script
    cd DedSec && bash Setup.sh

    The script will handle the complete installation. After the process, you will see a settings menu, you must choose Change Menu Style and then choose a menu style: list or grid.
    After that close Termux from notifications and reopen it.

Tip: After setup, you can quickly open the menu by typing e (for the English version) or g (for the Greek version) in Termux.

Understand What The DedSec Project Does

⚠️ Understanding Threats: Educational Demonstrations

CRITICAL NOTICE: The following scripts are included for educational and defensive purposes ONLY. Their function is to demonstrate how common attacks work, so you can learn to recognize and protect yourself against them. They should only be used in a controlled environment, on your own devices and accounts, for self-testing.

Toolkit Summary

  1. Fox Chat
    A secure, end-to-end encrypted chat application protected by a one-time Secret Key. Features include text messaging, voice notes, file sharing (up to 10 GB), live camera capture, and peer-to-peer video calls. Files shared in the chat can be downloaded directly from the chat interface by the participants.
  2. DedSec's Database
    A password-protected, self-hosted file storage server. It allows you to upload, download, search, and manage files through a secure web interface, automatically organizing them into categories like Documents, Images, and Videos. All files are stored in a Database folder created in the script's directory.
  3. OSINTDS
    A comprehensive tool for Open Source Intelligence (OSINT) gathering and web reconnaissance. It performs scans for WHOIS and DNS records, open ports, subdomains, and directories, and checks for common vulnerabilities like SQLi and XSS. It also includes an interactive HTML Inspector to download a full copy of a website for offline analysis. All reports and downloaded websites are saved in a dedicated folder inside [Your Downloads]/OSINTDS/.
  4. Phishing Demonstrations
    Modules that demonstrate how a malicious webpage can trick a user into giving away access to their device's camera, microphone, and location, or into entering personal details and card information. These scripts are for testing on your own devices to understand the importance of verifying links before clicking them.

    For your self-tests, any demonstration credentials or data you enter are saved locally into appropriately named folders inside your phone's main Downloads folder for you to review.

  5. URL Masker
    An educational tool to demonstrate how links can be disguised, helping you learn to identify potentially malicious URLs by showing how a seemingly innocent link can redirect to a different destination.
  6. Android App Launcher
    A utility to manage installed applications on your Android device. You can launch, view details for, uninstall, or extract the APK file of any app. It also includes an App Perm Inspector feature that scans the APK to identify dangerous permissions and detect embedded advertising trackers, generating a security report for your review.
  7. Settings
    A central control panel to manage the DedSec Project. Use it to view system information, update all project scripts and required packages, change the Termux prompt style, and switch between list or grid menu layouts. This script modifies your /data/data/com.termux/files/usr/etc/bash.bashrc file to apply changes.
  8. Loading Screen
    Installs a custom ASCII art loading screen that appears when you start Termux. You can use the default art, provide your own, and set the display duration. This script works by adding configuration to your ~/.bash_profile file.
  9. SSH Defender
    A Python-based SSH honeypot designed to detect and log unauthorized connection attempts. It intelligently cycles through common ports (like 22, 2222, 80, 443) and mimics a real SSH server to capture attacker data. Once an attack is detected on a port, it locks onto that port for permanent monitoring, logging all activity, IP addresses, and data sent by the attacker. It features a real-time TUI (Text-based User Interface) dashboard to monitor activity.
  10. Smart Notes
    A fast, TUI-based (Text-based User Interface) application for managing notes and reminders directly within the terminal. It allows you to create, edit, search, and delete notes, which are stored locally. It also features a built-in calendar system to set reminders, which can trigger Termux notifications at the scheduled time.
  11. Digital Footprint Finder
    An OSINT (Open Source Intelligence) tool used to discover the online presence of a username across hundreds of platforms. It performs a two-phase scan: first checking major social media and coding sites, and only proceeding to a full, broad scan if a match is found. Results are saved in both .txt and .json formats to your Downloads folder and are cached for 24 hours to speed up subsequent searches.
  12. Internet Tools
    A comprehensive, TUI-based suite of network tools for analysis and diagnostics. It integrates with the Termux:API to perform Wi-Fi scanning and management. Features include an internet speed test, local network device discovery, an NMAP-like port scanner with service detection, OS fingerprinting simulation, WHOIS/DNS lookups, an HTTP security header analyzer, and a web crawler, among other utilities.

Περίληψης Εργαλείων

  1. Fox Chat
    Μια ασφαλής, end-to-end κρυπτογραφημένη εφαρμογή συνομιλίας που προστατεύεται από ένα μυστικό κλειδί μίας χρήσης. Περιλαμβάνει ανταλλαγή μηνυμάτων κειμένου, φωνητικά μηνύματα, κοινή χρήση αρχείων (έως 10 GB), ζωντανή λήψη κάμερας και βιντεοκλήσεις peer-to-peer. Τα αρχεία που κοινοποιούνται μπορούν να ληφθούν απευθείας από τη διεπαφή συνομιλίας από τους συμμετέχοντες.
  2. DedSec's Database
    Ένας προστατευμένος με κωδικό πρόσβασης, αυτο-φιλοξενούμενος διακομιστής αποθήκευσης αρχείων. Σας επιτρέπει να ανεβάζετε, να κατεβάζετε, να αναζητάτε και να διαχειρίζεστε αρχεία μέσω μιας ασφαλούς διεπαφής ιστού, οργανώνοντάς τα αυτόματα σε κατηγορίες όπως Έγγραφα, Εικόνες και Βίντεο. Όλα τα ανεβασμένα αρχεία αποθηκεύονται σε έναν φάκελο Database που δημιουργείται στον κατάλογο του σεναρίου.
  3. OSINTDS
    Ένα ολοκληρωμένο εργαλείο για τη συλλογή πληροφοριών από ανοιχτές πηγές (OSINT) και την αναγνώριση ιστοσελίδων. Πραγματοποιεί σαρώσεις για εγγραφές WHOIS και DNS, ανοιχτές θύρες, subdomains και καταλόγους, και ελέγχει για κοινές ευπάθειες όπως SQLi και XSS. Περιλαμβάνει επίσης έναν διαδραστικό HTML Inspector για τη λήψη ενός πλήρους αντιγράφου μιας ιστοσελίδας για ανάλυση εκτός σύνδεσης. Όλες οι αναφορές και οι ληφθείσες ιστοσελίδες αποθηκεύονται σε έναν ειδικό φάκελο μέσα στις [Οι Λήψεις σας]/OSINTDS/.
  4. Επιδείξεις Phishing
    Modules που επιδεικνύουν πώς μια κακόβουλη ιστοσελίδα μπορεί να εξαπατήσει έναν χρήστη για να δώσει πρόσβαση στην κάμερα, το μικρόφωνο και την τοποθεσία της συσκευής του, ή για να εισάγει προσωπικά στοιχεία και στοιχεία κάρτας.

    Για τις δικές σας δοκιμές, τυχόν διαπιστευτήρια ή δεδομένα επίδειξης που εισάγετε αποθηκεύονται τοπικά σε φακέλους με κατάλληλα ονόματα μέσα στον κεντρικό φάκελο Λήψεις του τηλεφώνου σας για να τα ελέγξετε.

  5. URL Masker
    Ένα εκπαιδευτικό εργαλείο που επιδεικνύει πώς μπορούν να μεταμφιεστούν οι σύνδεσμοι, βοηθώντας σας να μάθετε να αναγνωρίζετε πιθανώς κακόβουλα URL, δείχνοντας πώς ένας φαινομενικά αθώος σύνδεσμος μπορεί να ανακατευθύνει σε διαφορετικό προορισμό.
  6. Android App Launcher
    Ένα βοηθητικό πρόγραμμα για τη διαχείριση των εγκατεστημένων εφαρμογών στη συσκευή σας Android. Μπορείτε να εκκινήσετε, να δείτε λεπτομέρειες, να απεγκαταστήσετε ή να εξαγάγετε το αρχείο APK οποιασδήποτε εφαρμογής. Περιλαμβάνει επίσης τη λειτουργία App Perm Inspector που σαρώνει το APK για να εντοπίσει επικίνδυνες άδειες και ενσωματωμένους trackers διαφημίσεων, δημιουργώντας μια αναφορά ασφαλείας για έλεγχο.
  7. Settings
    Ένας κεντρικός πίνακας ελέγχου για τη διαχείριση του DedSec Project. Χρησιμοποιήστε τον για να δείτε πληροφορίες συστήματος, να ενημερώσετε όλα τα σενάρια του έργου και τα απαιτούμενα πακέτα, να αλλάξετε το στυλ της γραμμής εντολών του Termux και να επιλέξετε μεταξύ διάταξης μενού λίστας ή πλέγματος. Αυτό το σενάριο τροποποιεί το αρχείο σας /data/data/com.termux/files/usr/etc/bash.bashrc για να εφαρμόσει τις αλλαγές.
  8. Loading Screen
    Εγκαθιστά μια προσαρμοσμένη οθόνη φόρτωσης με ASCII art που εμφανίζεται κατά την εκκίνηση του Termux. Μπορείτε να χρησιμοποιήσετε την προεπιλεγμένη εικόνα, να δώσετε τη δική σας και να ορίσετε τη διάρκεια εμφάνισης. Αυτό το σενάριο λειτουργεί προσθέτοντας ρυθμίσεις στο αρχείο σας ~/.bash_profile.
  9. SSH Defender
    Ένα honeypot SSH βασισμένο σε Python, σχεδιασμένο για τον εντοπισμό και την καταγραφή μη εξουσιοδοτημένων προσπαθειών σύνδεσης. Εναλλάσσεται έξυπνα μεταξύ κοινών θυρών (όπως 22, 2222, 80, 443) και μιμείται έναν πραγματικό διακομιστή SSH για να συλλέξει δεδομένα εισβολέων. Μόλις εντοπιστεί επίθεση σε μια θύρα, "κλειδώνει" σε αυτήν τη θύρα για μόνιμη παρακολούθηση, καταγράφοντας όλη τη δραστηριότητα, τις διευθύνσεις IP και τα δεδομένα που αποστέλλονται. Διαθέτει έναν πίνακα ελέγχου TUI (Text-based User Interface) πραγματικού χρόνου για την παρακολούθηση της δραστηριότητας.
  10. Smart Notes
    Μια γρήγορη εφαρμογή TUI (Text-based User Interface) για τη διαχείριση σημειώσεων και υπενθυμίσεων απευθείας στο τερματικό. Σας επιτρέπει να δημιουργείτε, να επεξεργάζεστε, να αναζητάτε και να διαγράφετε σημειώσεις, οι οποίες αποθηκεύονται τοπικά. Διαθέτει επίσης ενσωματωμένο σύστημα ημερολογίου για τη ρύθμιση υπενθυμίσεων, οι οποίες μπορούν να ενεργοποιήσουν ειδοποιήσεις Termux την προγραμματισμένη ώρα.
  11. Digital Footprint Finder
    Ένα εργαλείο OSINT (Open Source Intelligence) που χρησιμοποιείται για την ανακάλυψη της διαδικτυακής παρουσίας ενός ονόματος χρήστη σε εκατοντάδες πλατφόρμες. Πραγματοποιεί σάρωση δύο φάσεων: πρώτα ελέγχει μεγάλες πλατφόρμες κοινωνικής δικτύωσης και προγραμματισμού, και μόνο εάν βρεθεί αντιστοιχία, προχωρά σε πλήρη, ευρεία σάρωση. Τα αποτελέσματα αποθηκεύονται σε μορφές .txt και .json στον φάκελο Λήψεων και αποθηκεύονται προσωρινά για 24 ώρες για επιτάχυνση των επόμενων αναζητήσεων.
  12. Internet Tools
    Μια ολοκληρωμένη σουίτα εργαλείων δικτύου βασισμένη σε TUI για ανάλυση και διάγνωση. Ενσωματώνεται με το Termux:API για σάρωση και διαχείριση Wi-Fi. Οι λειτουργίες περιλαμβάνουν τεστ ταχύτητας διαδικτύου, ανακάλυψη συσκευών τοπικού δικτύου, σαρωτή θυρών τύπου NMAP με ανίχνευση υπηρεσιών, προσομοίωση αναγνώρισης λειτουργικού συστήματος, αναζητήσεις WHOIS/DNS, αναλυτή κεφαλίδων ασφαλείας HTTP και web crawler, μεταξύ άλλων.

Νομική Αποποίηση Ευθύνης: Όλα τα εργαλεία και οι πληροφορίες που παρουσιάζονται εδώ προορίζονται αποκλειστικά για εκπαιδευτικούς και ηθικούς ερευνητικούς σκοπούς. Χρησιμοποιήστε τα αποκλειστικά σε συστήματα που σας ανήκουν ή για τα οποία έχετε ρητή, γραπτή άδεια να δοκιμάσετε. Η μη εξουσιοδοτημένη χρήση είναι παράνομη. Ο χρήστης είναι αποκλειστικά υπεύθυνος για όλες τις ενέργειες που πραγματοποιεί, και οι δημιουργοί αυτού του έργου δεν αναλαμβάνουν καμία ευθύνη για οποιαδήποτε κακή χρήση ή ζημία.
