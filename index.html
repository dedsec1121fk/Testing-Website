How To Install And Setup The DedSec Project

Requirements

  • Device: Android with Termux installed.
  • Storage: Min 8GB free. (Images and recordings also consume space.)
  • RAM: Min 2GB.

Step-by-Step Installation

  1. Install Termux & Add-Ons

    Note: To install APKs (e.g., F-Droid), ensure you:
    - Enable unknown sources (Settings > Security > Install Unknown Apps).
    - Download F-Droid, then get Termux from F-Droid.
    - Install add-ons: Termux:API, Termux:Styling.
    - Allow the fdroid process when prompted.

  2. Update Packages & Install Git
    Important: Open the Termux application on your device before copying and pasting the command below.
    pkg update -y && pkg upgrade -y && pkg install git nano -y && termux-setup-storage
  3. Clone the Repository
    git clone https://github.com/dedsec1121fk/DedSec
  4. Run the Setup Script
    cd DedSec && bash Setup.sh

    The script will handle the complete installation. After the process, you will see a settings menu, you must choose Change Menu Style and then choose a menu style: list or grid.
    After that close Termux from notifications and reopen it.

Tip: After setup, you can quickly open the menu by typing e (for the English version) or g (for the Greek version) in Termux.

Basic Tools

Toolkit Summary

  1. Android App Launcher
    A utility to manage installed applications on your Android device. You can launch, view details for, uninstall, or extract the APK file of any app. It also includes an App Perm Inspector feature that scans the APK to identify dangerous permissions and detect embedded advertising trackers, generating a security report for your review.
  2. DedSec's Network
    A comprehensive network analysis and security toolkit that provides various network utilities including Wi-Fi scanning, port scanning, network discovery, speed tests, and security auditing. Features include passive Wi-Fi network analysis, enhanced port scanning with service detection, HTTP header security analysis, DNS record lookups, and various network diagnostic tools. All results and configuration files are saved in a dedicated folder named DedSec's Network inside the project directory for review.
  3. Digital Footprint Finder
    An OSINT (Open Source Intelligence) tool that helps you discover what public information exists about a username across multiple online platforms. It scans social media sites, coding platforms, and other services to find publicly accessible profiles associated with a username. The tool includes caching mechanisms to avoid repeated requests, stealth modes to reduce detection, and saves results in both text and JSON formats inside a Digital Footprint Finder folder in your phone's main Downloads directory for further analysis.
  4. File Converter
    An advanced, interactive file converter for Termux. It supports a wide range of document, image, audio/video, and data conversions (e.g., PDF, DOCX, PNG, MP4, CSV). All converted files are organized into a "File Converter" folder in your Downloads.
  5. Fox's Connections
    A secure, end-to-end encrypted chat application protected by a one-time Secret Key. Features include text messaging, voice notes, file sharing (up to 10 GB), live camera capture, and peer-to-peer video calls. Files shared in the chat can be downloaded directly from the chat interface by the participants. It also includes a public, link-based file database.
  6. QR Code Generator
    Generates a QR code image for any link you provide and saves it inside a 'QR Codes' folder within the phone's Downloads directory.
  7. Settings
    A central control panel to manage the DedSec Project. Use it to view system information, update all project scripts and required packages, change the Termux prompt style, and switch between list or grid menu layouts. This script modifies your /data/data/com.termux/files/usr/etc/bash.bashrc file to apply changes.
  8. Smart Notes
    A secure note-taking application with advanced features including encrypted storage, calendar integration, and reminder system. It provides a curses-based TUI interface for easy navigation, supports rich text editing, and includes a sophisticated search system. The reminder feature can schedule notifications using Termux's job scheduler, making it a powerful organizational tool.

Περίληψη Εργαλείων

  1. Android App Launcher
    Ένα βοηθητικό πρόγραμμα για τη διαχείριση των εγκατεστημένων εφαρμογών στη συσκευή σας Android. Μπορείτε να εκκινήσετε, να δείτε λεπτομέρειες, να απεγκαταστήσετε ή να εξαγάγετε το αρχείο APK οποιασδήποτε εφαρμογής. Περιλαμβάνει επίσης τη λειτουργία App Perm Inspector που σαρώνει το APK για να εντοπίσει επικίνδυνες άδειες και ενσωματωμένους trackers διαφημίσεων, δημιουργώντας μια αναφορά ασφαλείας για έλεγχο.
  2. DedSec's Network
    Ένα ολοκληρωμένο πακέτο εργαλείων ανάλυσης δικτύου και ασφάλειας που παρέχει διάφορα βοηθήματα δικτύου όπως σάρωση Wi-Fi, σάρωση θυρών, ανακάλυψη δικτύου, δοκιμές ταχύτητας και έλεγχο ασφαλείας. Περιλαμβάνει ανάλυση παθητικών δικτύων Wi-Fi, βελτιωμένη σάρωση θυρών με ανίχνευση υπηρεσιών, ανάλυση ασφαλείας κεφαλίδων HTTP, αναζήτηση εγγραφών DNS και διάφορα διαγνωστικά εργαλεία δικτύου. Όλα τα αποτελέσματα και τα αρχεία ρυθμίσεων αποθηκεύονται σε έναν ειδικό φάκελο με το όνομα DedSec's Network μέσα στον κατάλογο του έργου για ανασκόπηση.
  3. Digital Footprint Finder
    Ένα εργαλείο OSINT (Open Source Intelligence) που βοηθά στην ανακάλυψη δημόσιων πληροφοριών για ένα όνομα χρήστη σε πολλαπλές πλατφόρμες. Σαρώνει κοινωνικά δίκτυα, πλατφόρμες προγραμματισμού και άλλες υπηρεσίες για να βρει δημόσια προφίλ που σχετίζονται με ένα όνομα χρήστη. Το εργαλείο περιλαμβάνει μηχανισμούς προσωρινής αποθήκευσης, λειτουργίες stealth για μείωση της ανίχνευσης και αποθηκεύει τα αποτελέσματα σε μορφή κειμένου και JSON μέσα σε έναν φάκελο Digital Footprint Finder στον κύριο κατάλογο Λήψεις του τηλεφώνου σας για περαιτέρω ανάλυση.
  4. File Converter
    Ένας προηγμένος, διαδραστικός μετατροπέας αρχείων για το Termux. Υποστηρίζει ένα ευρύ φάσμα μετατροπών εγγράφων, εικόνων, ήχου/βίντεο και δεδομένων (π.χ. PDF, DOCX, PNG, MP4, CSV). Όλα τα μετατρεπόμενα αρχεία οργανώνονται σε έναν φάκελο "File Converter" στις Λήψεις σας.
  5. Fox's Connections
    Μια ασφαλής, end-to-end κρυπτογραφημένη εφαρμογή συνομιλίας που προστατεύεται από ένα μυστικό κλειδί μίας χρήσης. Περιλαμβάνει ανταλλαγή μηνυμάτων κειμένου, φωνητικά μηνύματα, κοινή χρήση αρχείων (έως 10 GB), ζωντανή λήψη κάμερας και βιντεοκλήσεις peer-to-peer. Τα αρχεία που κοινοποιούνται μπορούν να ληφθούν απευθείας από τη διεπαφή συνομιλίας από τους συμμετέχοντες. Περιλαμβάνει επίσης μια δημόσια βάση δεδομένων αρχείων προσβάσιμη με σύνδεσμο.
  6. QR Code Generator
    Δημιουργεί μια εικόνα QR code για οποιονδήποτε σύνδεσμο παρέχετε και την αποθηκεύει σε έναν φάκελο 'QR Codes' στον κατάλογο Λήψεων του τηλεφώνου.
  7. Settings
    Ένας κεντρικός πίνακας ελέγχου για τη διαχείριση του DedSec Project. Χρησιμοποιήστε τον για να δείτε πληροφορίες συστήματος, να ενημερώσετε όλα τα σενάρια του έργου και τα απαιτούμενα πακέτα, να αλλάξετε το στυλ της γραμμής εντολών του Termux και να επιλέξετε μεταξύ διάταξης μενού λίστας ή πλέγματος. Αυτό το σενάριο τροποποιεί το αρχείο σας /data/data/com.termux/files/usr/etc/bash.bashrc για να εφαρμόσει τις αλλαγές.
  8. Smart Notes
    Μια ασφαλής εφαρμογή σημειώσεων με προηγμένες λειτουργίες όπως κρυπτογραφημένη αποθήκευση, ενσωμάτωση ημερολογίου και σύστημα υπενθυμίσεων. Παρέχει διεπαφή TUI για εύκολη πλοήγηση, υποστηρίζει επεξεργασία πλούσιου κειμένου και περιλαμβάνει ένα εξελιγμένο σύστημα αναζήτησης.
