Installation & Setup

Requirements

  • Device: Android with Termux installed.
  • Storage: Min 3GB free. (The Radio feature requires more storage; images and recordings also consume space.)
  • RAM: Min 2GB.

Step-by-Step Installation

  1. Install Termux & Add-Ons

    Note: To install APKs (e.g., F-Droid), ensure you:
    - Enable unknown sources (Settings > Security > Install Unknown Apps).
    - Download F-Droid, then get Termux from F-Droid.
    - Install add-ons: Termux:API, Termux:Styling.
    - Allow the fdroid process when prompted.

  2. Run Setup Commands
    Important: Open the Termux application on your device before copying and pasting the commands below.
    termux-setup-storage && pkg update -y && pkg upgrade -y && pkg install aapt clang cloudflared curl ffmpeg fzf git jq libffi libffi-dev libxml2 libxslt nano ncurses nodejs openssh openssl openssl-tool proot python rust unzip wget zip -y
    pip install blessed bs4 cryptography flask flask-socketio geopy mutagen phonenumbers pycountry pydub pycryptodome requests werkzeug
    pkg install termux-api
  3. Clone Repository
    git clone https://github.com/dedsec1121fk/DedSec

    Tip: If you get an error like "ModuleNotFoundError: No module named 'requests'", use this command and try again:

    pip install requests
  4. Choose Menu Style
    cd DedSec/Scripts && python Settings.py

    Tip: You can open the menu later by typing m in Termux.

Portable Laptop

Requirements

  • Device: Android (non-rooted).
  • Storage: Minimum 4GB free.
  • RAM: Minimum 3GB (4GB+ recommended for a smooth experience).

Required Android Apps

  • Termux: The base terminal environment. Must be installed from F-Droid, as the Play Store version is outdated.
  • VNC Viewer: An app to connect to and control your graphical desktop.

All software runs inside a container and does not require your device to be rooted.

Master Command #1: Complete Desktop Setup

Note: This command does everything to set up your base desktop. It will take 20-50 minutes and requires a stable Wi-Fi connection.

pkg update -y && pkg upgrade -y && pkg install proot-distro -y && proot-distro install debian && proot-distro login debian -- "apt update && apt upgrade -y && apt install xfce4 xfce4-goodies tigervnc-standalone-server nano -y"

Step-by-Step System Setup

  1. Set VNC Password (One-Time Only)

    After Master Command #1 finishes, run these commands in Termux:

    proot-distro login debian
    vncserver

    Enter a 6-8 character password. Type 'n' for the view-only password.

  2. Configure VNC Server (One-Time Only)

    Still inside Debian, run these commands:

    vncserver -kill :1
    nano ~/.vnc/xstartup

    Delete all text in the file and replace it with:
    #!/bin/bash
    startxfce4 &
    Save with Ctrl+X, then Y, then Enter.

  3. Start & Stop Your Desktop

    To START, run this inside Debian:

    vncserver -geometry 1280x720

    Then connect with a VNC Viewer app to localhost:1.

    To STOP, run this inside Debian:

    vncserver -kill :1

Master Command #2: Install All Laptop Software

After starting your desktop, open the Terminal Emulator inside it to install a full suite of useful applications. Here is what's included:

  • synaptic: A graphical "App Store" to easily find and install thousands of other programs.
  • firefox-esr: A full-featured, stable version of the Firefox web browser.
  • libreoffice: A complete office suite (like Microsoft Office) for documents, spreadsheets, and presentations.
  • gimp: A powerful image editor (like Photoshop) for photo retouching and graphic design.
  • vlc: A versatile media player that can play almost any video or audio file.
  • geany: A lightweight and fast code editor and IDE for programming.
  • htop: A system monitor to see how much CPU and RAM your apps are using.
  • thunderbird: A desktop email client to manage your emails from multiple accounts.
  • pidgin: A universal chat client that connects to multiple messaging networks.
  • transmission-gtk: An easy-to-use client for downloading torrents.
  • zim: A personal wiki/notebook for organizing notes and ideas.
  • evince: A simple document viewer for reading PDFs.
  • foliate: A modern and clean e-book reader for `.epub` files.
  • inkscape: A professional vector graphics editor (like Adobe Illustrator).
  • scribus: A desktop publishing app for creating magazines and flyers.
  • pinta: A simple drawing and image editing program, easier than GIMP for quick edits.
  • engrampa: A tool to create and open compressed files like `.zip` and `.rar`.
  • baobab: A disk usage analyzer that visually shows what's taking up space.
  • gufw: An easy-to-use graphical interface for managing the system's firewall.

Βήμα-Βήμα Ρύθμιση Συστήματος

  1. Ορισμός Κωδικού Πρόσβασης VNC (Μόνο για μία φορά)

  2. Διαμόρφωση του VNC Server (Μόνο για μία φορά)

    Διαγράψτε όλο το κείμενο στο αρχείο και αντικαταστήστε το με:

    Αποθηκεύστε με Ctrl+X, μετά Y, και μετά Enter.

  3. Εκκίνηση & Τερματισμός της Επιφάνειας Εργασίας σας

    Για ΕΚΚΙΝΗΣΗ, εκτελέστε αυτό μέσα στο Debian:

    vncserver -geometry 1280x720

    Στη συνέχεια, συνδεθείτε με μια εφαρμογή VNC Viewer στη διεύθυνση localhost:1.

    Για ΤΕΡΜΑΤΙΣΜΟ, εκτελέστε αυτό μέσα στο Debian:

    vncserver -kill :1

  • htop: Ένας παρακολούθησης συστήματος για να βλέπετε πόση CPU και RAM χρησιμοποιούν οι εφαρμογές σας.

Digital Self-Defense Toolkit

⚠️ Understanding Threats: Educational Demonstrations

CRITICAL NOTICE: The following scripts are included for educational and defensive purposes ONLY. Their function is to demonstrate how common attacks work, so you can learn to recognize and protect yourself against them. They should only be used in a controlled environment, on your own devices and accounts, for self-testing.

Toolkit Summary

  1. Fox Chat & Utilities
    Includes various scripts like encrypted chat, an offline radio player, app management, and the settings menu to update the project.
  2. Phishing Demonstrations
    Modules that demonstrate how a malicious webpage can trick a user into giving away access to their device's camera, microphone, and location. These scripts are for testing on your own devices to understand the importance of verifying links before clicking them. The toolkit contains various proof-of-concept pages that simulate common social media and service logins to show how attackers create convincing fake pages.

    For your self-tests, any demonstration credentials or data you enter are saved locally into appropriately named folders inside your phone's main Downloads folder for you to review.

  3. SocialBrute
    A script designed to demonstrate the weakness of simple or reused passwords. It shows how automated attacks work, highlighting the critical importance of using strong, unique passwords for every online account. When testing, any results are saved locally in the Downloads folder.
  4. URL Masker & HTML Finder
    Educational tools to help you understand how web pages are structured and how links can be disguised. Any pages downloaded by HTML Finder are saved to the Downloads folder.

Πρέπει να χρησιμοποιούνται μόνο σε ελεγχόμενο περιβάλλον, στις δικές σας συσκευές και λογαριασμούς, για αυτο-έλεγχο.

Περίληψη Εργαλείων

  1. Fox Chat & Βοηθητικά προγράμματα
    Περιλαμβάνει διάφορα βοηθητικά σενάρια όπως κρυπτογραφημένη συνομιλία, ένα ραδιόφωνο εκτός σύνδεσης, εργαλεία διαχείρισης εφαρμογών και το μενού ρυθμίσεων για την ενημέρωση του έργου.
  2. Επιδείξεις Phishing
    Modules που επιδεικνύουν πώς μια κακόβουλη ιστοσελίδα μπορεί να εξαπατήσει έναν χρήστη για να δώσει πρόσβαση στην κάμερα, το μικρόφωνο και την τοποθεσία της συσκευής του. Αυτά τα σενάρια είναι για δοκιμή στις δικές σας συσκευές για να κατανοήσετε τη σημασία της επαλήθευσης των συνδέσμων πριν κάνετε κλικ. Η εργαλειοθήκη περιέχει διάφορες σελίδες απόδειξης της ιδέας (proof-of-concept) που προσομοιώνουν συνήθεις συνδέσεις σε μέσα κοινωνικής δικτύωσης και υπηρεσίες για να δείξουν πώς οι επιτιθέμενοι δημιουργούν πειστικές ψεύτικες σελίδες.

    Για τις δικές σας δοκιμές, τυχόν διαπιστευτήρια ή δεδομένα επίδειξης που εισάγετε αποθηκεύονται τοπικά σε φακέλους με κατάλληλα ονόματα μέσα στον κεντρικό φάκελο Λήψεις του τηλεφώνου σας για να τα ελέγξετε.

  3. SocialBrute
    Ένα σενάριο σχεδιασμένο για να επιδεικνύει την αδυναμία των απλών ή επαναχρησιμοποιημένων κωδικών πρόσβασης. Δείχνει πώς λειτουργούν οι αυτοματοποιημένες επιθέσεις, τονίζοντας την κρίσιμη σημασία της χρήσης ισχυρών, μοναδικών κωδικών πρόσβασης για κάθε online λογαριασμό. Κατά τη δοκιμή, τυχόν αποτελέσματα αποθηκεύονται τοπικά στον φάκελο Λήψεις.
  4. URL Masker & HTML Finder
    Εκπαιδευτικά εργαλεία για να σας βοηθήσουν να κατανοήσετε πώς δομούνται οι ιστοσελίδες και πώς μπορούν να μεταμφιεστούν οι σύνδεσμοι. Οποιεσδήποτε σελίδες λαμβάνονται από το HTML Finder αποθηκεύονται στον φάκελο Λήψεις.
