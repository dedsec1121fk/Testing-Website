How To Install And Setup The DedSec Project

Requirements

  • Device: Android with Termux installed.
  • Storage: Min 8GB free. (Images and recordings also consume space.)
  • RAM: Min 2GB.

Step-by-Step Installation

  1. Install Termux & Add-Ons

    Note: To install APKs (e.g., F-Droid), ensure you:
    - Enable unknown sources (Settings > Security > Install Unknown Apps).
    - Download F-Droid, then get Termux from F-Droid.
    - Install add-ons: Termux:API, Termux:Styling.
    - Allow the fdroid process when prompted.

  2. Update Packages & Install Git
    Important: Open the Termux application on your device before copying and pasting the command below.
    pkg update -y && pkg upgrade -y && pkg install git nano -y && termux-setup-storage
  3. Clone the Repository
    git clone https://github.com/dedsec1121fk/DedSec
  4. Run the Setup Script
    cd DedSec && bash Setup.sh

    The script will handle the complete installation. After the process, you will see a settings menu, you must choose Change Menu Style and then choose a menu style: list or grid.
    After that close Termux from notifications and reopen it.

Tip: After setup, you can quickly open the menu by typing e in Termux.

CRITICAL NOTICE: The following scripts are included for educational and defensive purposes ONLY. Their function is to demonstrate how common attacks work, so you can learn to recognize and protect yourself against them. They should only be used in a controlled environment, on your own devices and accounts, for self-testing.

Toolkit Summary

  1. Fox Chat
    A secure chat application with full UI and support for 10GB uploads. It runs from a single server along with DedSec's Database using one secret key.
  2. DedSec's Database
    A password-protected, self-hosted file storage server that runs on the same single server as Fox Chat.
  3. Advanced Network & Security Toolkit
    An optimized and enhanced networking and security toolset designed specifically for Termux. It is built to run 100% without root access and is efficient for devices with low resources (e.g., 2GB RAM).
  4. Digital Footprint Finder
    An OSINT tool that finds public accounts for a username across multiple platforms, with an option for Stealth Mode. It includes a 24-hour search cache and saves results as .txt and .json files.
  5. Android App Launcher
    A utility for managing installed Android applications. Features include extracting the APK file to your storage and an App Perm Inspector feature to analyze application security permissions.
  6. Termux File Converter
    An advanced, interactive file converter using a curses TUI (Text User Interface). It supports many data, document, image, and A/V conversions, organizing files into 40 sub-folders located in /storage/emulated/0/Download/File Converter/.
  7. QR Code Generator
    A utility to create a QR code image for any link (URL). The generated image file is saved inside a dedicated 'QR Codes' folder in your storage.
  8. Settings
    The central control panel to update the DedSec Project, update packages, and manage the environment. Options include changing the Termux prompt style and switching between list or grid menu layouts by modifying bash.bashrc.
  9. Smart Notes
    A secure note-taking application featuring a curses-based TUI. It saves notes in JSON format and includes command-line functions to add notes and run reminders.
  10. Extra Content Extractor
    A utility to easily copy the entire 'Extra Content' folder (which typically holds wallpapers or other media) from the project directory to your main phone downloads folder (~/storage/downloads/Extra Content).
  11. Custom Loading Screen
    A utility to install a personalized **ASCII-Art loading screen** in Termux by modifying the `bash_profile`. It allows the user to set a custom delay and use their own ASCII art.
  12. URL Masker
    An educational tool to demonstrate how links can be disguised, helping you learn to identify potentially malicious URLs by showing how a seemingly innocent link can redirect to a different destination.
  13. Social Media Fake Pages Scripts
    Scripts that can create fake pages for famous Social Medias and not only. Their saves can be found in folders inside your internal storage Downloads folder.
  14. Personal Information Capture Scripts
    Scripts that can catch camera (front or back) photos, microphone recordings, location and not only. Their saves can be found in folders inside your internal storage Downloads folder.
