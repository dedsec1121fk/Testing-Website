How To Install And Setup The DedSec Project

Requirements

  • Device: Android with Termux installed.
  • Storage: Min 8GB free.
  • RAM: Min 2GB.

Step-by-Step Installation

  1. Install Termux & Add-Ons

    Note: To install APKs (e.g., F-Droid), ensure you:
    - Enable unknown sources (Settings > Security > Install Unknown Apps).
    - Download F-Droid, then get Termux from F-Droid.
    - Install add-ons: Termux:API, Termux:Styling.
    - Allow the fdroid process when prompted.

  2. Update Packages & Install Git
    Important: Open the Termux application on your device before copying and pasting the command below.
    pkg update -y && pkg upgrade -y && pkg install git nano -y && termux-setup-storage
  3. Clone the Repository
    git clone https://github.com/dedsec1121fk/DedSec
  4. Run the Setup Script
    cd DedSec && bash Setup.sh

    The script will handle the complete installation. After the process, you will see a settings menu, you must choose Change Menu Style and then choose a menu style: list or grid.
    After that close Termux from notifications and reopen it.

Tip: After setup, you can quickly open the menu by typing e (for the English version) or g (for the Greek version) in Termux.

Understand What The DedSec Project Does

⚠️ Understanding Threats: Educational Demonstrations

CRITICAL NOTICE: The following scripts are included for educational and defensive purposes ONLY. Their function is to demonstrate how common attacks work, so you can learn to recognize and protect yourself against them. They should only be used in a controlled environment, on your own devices and accounts, for self-testing.

Toolkit Summary

  1. Fox Chat
    A secure, end-to-end encrypted chat application protected by a one-time Secret Key. Features include text messaging, voice notes, file sharing (up to 10 GB), live camera capture, and peer-to-peer video calls. Files shared in the chat can be downloaded directly from the chat interface by the participants.
  2. DedSec's Database
    A password-protected, self-hosted file storage server. It allows you to upload, download, search, and manage files through a secure web interface, automatically organizing them into categories like Documents, Images, and Videos. All files are stored in a Database folder created in the script’s directory.
  3. OSINTDS
    A comprehensive tool for Open Source Intelligence (OSINT) gathering and web reconnaissance. It performs scans for WHOIS and DNS records, open ports, subdomains, and directories, and checks for common vulnerabilities like SQLi and XSS. It also includes an interactive HTML Inspector to download a full copy of a website for offline analysis. All reports and downloaded websites are saved in a dedicated folder inside [Your Downloads]/OSINTDS/.
  4. Phishing Demonstrations
    Modules that demonstrate how a malicious webpage can trick a user into giving away access to their device's camera, microphone, and location, or into entering personal details and card information. These scripts are for testing on your own devices to understand the importance of verifying links before clicking them.

    For your self-testing environment, the project uses publicly available images for its demo phishing page to save storage. The tool requires a secure network environment for proper use and accurate reporting.

Περίληψη Εργαλειοθήκης

  1. Fox Chat
    Μια ασφαλής, κρυπτογραφημένη από άκρο σε άκρο εφαρμογή συνομιλίας που προστατεύεται από ένα μοναδικό Μυστικό Κλειδί. Οι λειτουργίες περιλαμβάνουν γραπτά μηνύματα, φωνητικές σημειώσεις, κοινή χρήση αρχείων (έως 10 GB), ζωντανή λήψη κάμερας και βιντεοκλήσεις peer-to-peer. Τα αρχεία που κοινοποιούνται στη συνομιλία μπορούν να ληφθούν απευθείας από τη διεπαφή συνομιλίας από τους συμμετέχοντες.
  2. DedSec's Database
    Ένας προστατευμένος με κωδικό πρόσβασης, αυτο-φιλοξενούμενος διακομιστής αποθήκευσης αρχείων. Σας επιτρέπει να ανεβάζετε, να κατεβάζετε, να αναζητάτε και να διαχειρίζεστε αρχεία μέσω μιας ασφαλούς διεπαφής Ιστού, οργανώνοντάς τα αυτόματα σε κατηγορίες όπως Έγγραφα, Εικόνες και Βίντεο. Όλα τα αρχεία αποθηκεύονται σε έναν φάκελο Database που δημιουργείται στον κατάλογο του σεναρίου.
  3. OSINTDS
    Ένα ολοκληρωμένο εργαλείο για τη συλλογή πληροφοριών ανοιχτής πηγής (OSINT) και την αναγνώριση Ιστού. Πραγματοποιεί σαρώσεις για WHOIS και DNS εγγραφές, ανοιχτές θύρες, υποτομείς και καταλόγους, και ελέγχει για κοινές ευπάθειες όπως SQLi και XSS. Περιλαμβάνει επίσης έναν διαδραστικό επιθεωρητή HTML για τη λήψη ενός πλήρους αντιγράφου ενός ιστότοπου για ανάλυση εκτός σύνδεσης. Όλες οι αναφορές και οι ληφθέντες ιστότοποι αποθηκεύονται σε έναν ειδικό φάκελο μέσα στο [Οι Λήψεις σας]/OSINTDS/.
  4. Phishing Demonstrations
    Ενότητες που δείχνουν πώς μια κακόβουλη ιστοσελίδα μπορεί να ξεγελάσει έναν χρήστη ώστε να δώσει πρόσβαση στην κάμερα, το μικρόφωνο και την τοποθεσία της συσκευής του, ή να εισάγει προσωπικά στοιχεία και πληροφορίες κάρτας. Αυτά τα σενάρια είναι για δοκιμή στις δικές σας συσκευές για να κατανοήσετε τη σημασία της επαλήθευσης των συνδέσμων πριν κάνετε κλικ σε αυτούς.

    Για το περιβάλλον αυτο-δοκιμής σας, το έργο χρησιμοποιεί δημόσια διαθέσιμες εικόνες για τη σελίδα επίδειξης phishing για εξοικονόμηση αποθηκευτικού χώρου. Το εργαλείο απαιτεί ένα ασφαλές περιβάλλον δικτύου για σωστή χρήση και ακριβή αναφορά.

Η Fox CyberSec είναι μια ομάδα ειδικών στον κυβερνοχώρο που επικεντρώνεται στην παροχή ισχυρών αμυντικών λύσεων και συμβουλών ηθικού hacking. Η συνεργασία μας εστιάζει στην αξιολόγηση των επιδείξεων εκμετάλλευσης για εκπαιδευτική ακρίβεια και στη διασφάλιση ότι όλα τα εργαλεία ευθυγραμμίζονται με τις ηθικές οδηγίες.

Επισκεφτείτε το GitHub

Phantom Sec

Η Phantom Sec ειδικεύεται σε προηγμένες δοκιμές διείσδυσης και την ασφάλεια δικτυακής υποδομής. Συμβάλλουν στο έργο μας παρέχοντας πλαίσιο πραγματικού κόσμου και επικυρώνοντας την αποτελεσματικότητα των εκπαιδευτικών ενοτήτων μας στην προσομοίωση σύνθετων απειλών δικτύου.

Επισκεφτείτε το GitHub

Ghost Labs

Η Ghost Labs επικεντρώνεται σε τεχνολογίες διατήρησης της ιδιωτικής ζωής και ασφαλή πρωτόκολλα επικοινωνίας. Βοηθούν στη διασφάλιση ότι το εργαλείο Fox Chat διατηρεί τα πρότυπα κρυπτογράφησης και την ιδιωτική ζωή αιχμής από τη σχεδίαση.

Our tools are provided as open-source software. They do not send data back to us. Any data generated or stored by the tools (e.g., chat logs, database files, OSINT reports) is stored locally on the user's device. We do not have access to, or responsibility for, this local data.

4. Children’s Privacy

Our Service is not intended for use by anyone under the age of 13. We do not knowingly collect PII from children under 13.

5. Changes to This Privacy Policy

We may update our Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page. These changes are effective immediately after they are posted.

6. Contact Us

If you have any questions or suggestions about our Privacy Policy, do not hesitate to contact us via the methods provided on the "Contact Us" page.

Ημερομηνία Ισχύος: 31 Οκτωβρίου 2024

Το DedSec Project («εμείς») δεσμεύεται να προστατεύει το απόρρητό σας. Αυτή η πολιτική περιγράφει τον τρόπο με τον οποίο χειριζόμαστε τις πληροφορίες σε σχέση με τον ιστότοπο και το λογισμικό μας.

1. Πληροφορίες που δεν συλλέγουμε

Δεν χρησιμοποιούμε cookies παρακολούθησης, αναλυτικά στοιχεία ή συλλέγουμε προσωπικά αναγνωρίσιμα στοιχεία (PII) από τους επισκέπτες του ιστότοπού μας ή τους χρήστες των εργαλείων ανοιχτού κώδικα. Το όνομά σας, η διεύθυνση IP και τα δεδομένα χρήσης δεν καταγράφονται ή αποθηκεύονται από εμάς.

2. Υπηρεσίες Τρίτων

Ο ιστότοπός μας ενδέχεται να χρησιμοποιεί υπηρεσίες τρίτων, όπως το Google AdSense, οι οποίες ενδέχεται να χρησιμοποιούν cookies για διαφημιστικούς σκοπούς. Σας συνιστούμε ανεπιφύλακτα να ελέγξετε τις πολιτικές απορρήτου τυχόν υπηρεσιών τρίτων για πληροφορίες σχετικά με τη συλλογή δεδομένων και τις πρακτικές χρήσης τους. Δεν είμαστε υπεύθυνοι για τις πρακτικές απορρήτου αυτών των εξωτερικών ιστότοπων.

3. Εργαλεία Ανοιχτού Κώδικα

Τα εργαλεία μας παρέχονται ως λογισμικό ανοιχτού κώδικα. Δεν μας στέλνουν δεδομένα. Οποιαδήποτε δεδομένα δημιουργούνται ή αποθηκεύονται από τα εργαλεία (π.χ., αρχεία καταγραφής συνομιλίας, αρχεία βάσης δεδομένων, αναφορές OSINT) αποθηκεύονται τοπικά στη συσκευή του χρήστη. Δεν έχουμε πρόσβαση ή ευθύνη για αυτά τα τοπικά δεδομένα.

4. Προστασία Προσωπικών Δεδομένων Παιδιών

Η Υπηρεσία μας δεν προορίζεται για χρήση από οποιονδήποτε κάτω των 13 ετών. Δεν συλλέγουμε εν γνώσει μας PII από παιδιά κάτω των 13 ετών.

5. Αλλαγές σε Αυτήν την Πολιτική Απορρήτου

Ενδέχεται να ενημερώνουμε την Πολιτική Απορρήτου μας κατά καιρούς. Θα σας ειδοποιήσουμε για τυχόν αλλαγές δημοσιεύοντας τη νέα Πολιτική Απορρήτου σε αυτή τη σελίδα. Αυτές οι αλλαγές ισχύουν αμέσως μετά τη δημοσίευσή τους.

6. Επικοινωνήστε μαζί μας

Εάν έχετε οποιεσδήποτε ερωτήσεις ή προτάσεις σχετικά με την Πολιτική Απορρήτου μας, μη διστάσετε να επικοινωνήσετε μαζί μας μέσω των μεθόδων που παρέχονται στη σελίδα «Επικοινωνία».
