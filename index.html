How To Install And Setup The DedSec Project

Requirements

  • Device: Android with Termux installed.
  • Storage: Min 8GB free. (Images and recordings also consume space.)
  • RAM: Min 2GB.

Step-by-Step Installation

  1. Install Termux & Add-Ons

    Note: To install APKs (e.g., F-Droid), ensure you:
    - Enable unknown sources (Settings > Security > Install Unknown Apps).
    - Download F-Droid, then get Termux from F-Droid.
    - Install add-ons: Termux:API, Termux:Styling.
    - Allow the fdroid process when prompted.

  2. Update Packages & Install Git
    Important: Open the Termux application on your device before copying and pasting the command below.
    pkg update -y && pkg upgrade -y && pkg install git nano -y && termux-setup-storage
  3. Clone the Repository
    git clone https://github.com/dedsec1121fk/DedSec
  4. Run the Setup Script
    cd DedSec && bash Setup.sh

    The script will handle the complete installation. After the process, you will see a settings menu, you must choose Change Menu Style and then choose a menu style: list or grid.
    After that close Termux from notifications and reopen it.

Tip: After setup, you can quickly open the menu by typing e (for the English version) or g (for the Greek version) in Termux.

Understand What The DedSec Project Does

⚠️ Understanding Threats: Educational Demonstrations

CRITICAL NOTICE: The following scripts are included for educational and defensive purposes ONLY. Their function is to demonstrate how common attacks work, so you can learn to recognize and protect yourself against them. They should only be used in a controlled environment, on your own devices and accounts, for self-testing.

Toolkit Summary

  1. Fox's Connections (Chat & Database)
    Provides a secure, end-to-end encrypted chat application (Fox Chat) with messaging, voice notes, large file sharing (up to 10GB), and video calls. It also includes a self-hosted file storage server (DedSec's Database) accessible via web interface for uploading, downloading, and managing files organized into categories. Both run from a single server process with one secret key.
  2. DedSec's Network (Internet Tools)
    A comprehensive network analysis and security toolkit. Includes Wi-Fi scanning (no root), Nmap wrapper, enhanced Python-based port scanning, network discovery, OSINTDS web reconnaissance (WHOIS, DNS, vuln checks, HTML Inspector), SSH Defender honeypot, internet speed test, IP info, and network diagnostics (Ping/Traceroute). Results and configurations are saved in the 'DedSec's Network' folder.
  3. Phishing Demonstrations
    Modules that demonstrate how a malicious webpage can trick a user into giving away access to their device's camera, microphone, and location, or into entering personal details and card information. These scripts are for testing on your own devices to understand the importance of verifying links before clicking them.

    For your self-tests, any demonstration credentials or data you enter are saved locally into appropriately named folders inside your phone's main Downloads folder for you to review.

  4. Social Media Fake Pages
    Scripts that can create fake pages for famous Social Medias and not only. Their saves can be found in folders inside your internal storage Downloads folder.
  5. Personal Information Capture
    Scripts that can catch camera (front or back) photos, microphone recordings, location and not only. Their saves can be found in folders inside your internal storage Downloads folder.
  6. URL Masker
    An educational tool demonstrating how links can be disguised using shortening services like is.gd (with custom alias) or cleanuri.com (as fallback). Helps users learn to identify potentially malicious URLs.
  7. Android App Launcher
    A utility to manage installed Android applications. Allows launching, viewing details, uninstalling, or extracting APK files. Features an 'App Perm Inspector' to scan APKs for dangerous permissions and advertising trackers, generating a security report saved to the Downloads/App_Security_Reports folder.
  8. Settings
    A central control panel to manage the DedSec Project. Use it to view system information, update all project scripts and required packages (Termux & Python), change the Termux prompt style, and switch between list or grid menu layouts. It modifies bash configuration files (bash.bashrc) to apply changes and manage startup behavior.
  9. Loading Screen
    Installs a custom ASCII art loading screen that appears when you start Termux, using either default or user-provided art. Allows setting the display duration in seconds. Modifies the ~/.bash_profile file to implement the screen.
  10. Digital Footprint Finder
    An OSINT tool to discover public profiles associated with a username across various online platforms (social media, coding sites, etc.). Features caching (24-hour default), stealth modes (less aggressive scanning), and saves results in text and JSON formats to a 'Digital Footprint Finder' folder in Downloads.
  11. Smart Notes
    A secure, terminal-based (TUI) note-taking application using curses. Features encrypted storage (in ~/.smart_notes.json), calendar integration, reminders (using Termux job scheduler via dateutil parsing and optional command execution), rich text support, and advanced search capabilities.
  12. QR Code Generator
    A simple tool to generate QR code images from text (typically URLs). Saves the generated PNG image to a 'QR Codes' folder within the main Downloads directory, using a sanitized version of the input data as the filename.
  13. File Converter
    An interactive, terminal-based (TUI) file converter supporting various formats across documents, images (including SVG/PSD via cairosvg/psd-tools), audio/video (requires ffmpeg), archives (ZIP, RAR via unrar, 7Z, TAR, GZ), and data (CSV/JSON). Organizes converted files into 40 categorized sub-folders within a main 'File Converter' directory in Downloads.
  14. Extra Content
    A utility script that copies the 'Extra Content' folder (and all its contents) from the DedSec project directory (`~/DedSec/Extra Content`) to the main Downloads folder (`~/storage/downloads/Extra Content`) on the device storage.

Περίληψης Εργαλείων (Απαιτείται Μετάφραση)

  1. Fox's Connections (Chat & Database)
    Παρέχει μια ασφαλή, end-to-end κρυπτογραφημένη εφαρμογή συνομιλίας (Fox Chat) με μηνύματα, φωνητικές σημειώσεις, κοινή χρήση μεγάλων αρχείων (έως 10GB) και βιντεοκλήσεις. Περιλαμβάνει επίσης έναν αυτο-φιλοξενούμενο διακομιστή αποθήκευσης αρχείων (DedSec's Database) προσβάσιμο μέσω web interface για ανέβασμα, κατέβασμα και διαχείριση αρχείων οργανωμένων σε κατηγορίες. Και τα δύο εκτελούνται από μια ενιαία διαδικασία διακομιστή με ένα μυστικό κλειδί.
  2. DedSec's Network (Internet Tools)
    Ένα ολοκληρωμένο πακέτο εργαλείων ανάλυσης δικτύου και ασφάλειας. Περιλαμβάνει σάρωση Wi-Fi (χωρίς root), περιτύλιγμα Nmap, βελτιωμένη σάρωση θυρών βασισμένη σε Python, ανακάλυψη δικτύου, αναγνώριση ιστού OSINTDS (WHOIS, DNS, έλεγχοι ευπαθειών, HTML Inspector), honeypot SSH Defender, τεστ ταχύτητας internet, πληροφορίες IP και διαγνωστικά δικτύου (Ping/Traceroute). Τα αποτελέσματα και οι ρυθμίσεις αποθηκεύονται στον φάκελο 'DedSec's Network'.
  3. Phishing Demonstrations
    Modules που επιδεικνύουν πώς μια κακόβουλη ιστοσελίδα μπορεί να εξαπατήσει έναν χρήστη για να δώσει πρόσβαση στην κάμερα, το μικρόφωνο και την τοποθεσία της συσκευής του, ή για να εισάγει προσωπικά στοιχεία και στοιχεία κάρτας. Αυτά τα σενάρια προορίζονται για δοκιμή στις δικές σας συσκευές για να κατανοήσετε τη σημασία της επαλήθευσης των συνδέσμων πριν κάνετε κλικ.

    Για τις δικές σας δοκιμές, τυχόν διαπιστευτήρια ή δεδομένα επίδειξης που εισάγετε αποθηκεύονται τοπικά σε φακέλους με κατάλληλα ονόματα μέσα στον κεντρικό φάκελο Λήψεις του τηλεφώνου σας για να τα ελέγξετε.

  4. Social Media Fake Pages
    Scripts που μπορούν να δημιουργήσουν ψεύτικες σελίδες για διάσημα Κοινωνικά Δίκτυα και όχι μόνο. Οι αποθηκεύσεις τους βρίσκονται σε φακέλους μέσα στον φάκελο Λήψεις της εσωτερικής σας αποθήκευσης.
  5. Personal Information Capture
    Scripts που μπορούν να καταγράψουν φωτογραφίες από κάμερα (μπροστινή ή πίσω), ηχογραφήσεις μικροφώνου, τοποθεσία και όχι μόνο. Οι αποθηκεύσεις τους βρίσκονται σε φακέλους μέσα στον φάκελο Λήψεις της εσωτερικής σας αποθήκευσης.
  6. URL Masker
    Ένα εκπαιδευτικό εργαλείο που επιδεικνύει πώς μπορούν να μεταμφιεστούν οι σύνδεσμοι χρησιμοποιώντας υπηρεσίες συντόμευσης όπως το is.gd (με προσαρμοσμένο ψευδώνυμο) ή το cleanuri.com (ως εναλλακτική). Βοηθά τους χρήστες να μάθουν να αναγνωρίζουν πιθανώς κακόβουλα URL.
  7. Android App Launcher
    Ένα βοηθητικό πρόγραμμα για τη διαχείριση των εγκατεστημένων εφαρμογών Android. Επιτρέπει την εκκίνηση, την προβολή λεπτομερειών, την απεγκατάσταση ή την εξαγωγή αρχείων APK. Διαθέτει έναν 'App Perm Inspector' για τη σάρωση APK για επικίνδυνες άδειες και trackers διαφημίσεων, δημιουργώντας μια αναφορά ασφαλείας που αποθηκεύεται στον φάκελο Downloads/App_Security_Reports.
  8. Settings
    Ένας κεντρικός πίνακας ελέγχου για τη διαχείριση του DedSec Project. Χρησιμοποιήστε τον για να δείτε πληροφορίες συστήματος, να ενημερώσετε όλα τα σενάρια του έργου και τα απαιτούμενα πακέτα (Termux & Python), να αλλάξετε το στυλ της γραμμής εντολών του Termux και να επιλέξετε μεταξύ διάταξης μενού λίστας ή πλέγματος. Τροποποιεί αρχεία διαμόρφωσης bash (bash.bashrc) για να εφαρμόσει τις αλλαγές και να διαχειριστεί τη συμπεριφορά εκκίνησης.
  9. Loading Screen
    Εγκαθιστά μια προσαρμοσμένη οθόνη φόρτωσης με ASCII art που εμφανίζεται κατά την εκκίνηση του Termux, χρησιμοποιώντας είτε προεπιλεγμένη είτε παρεχόμενη από τον χρήστη τέχνη. Επιτρέπει τον ορισμό της διάρκειας εμφάνισης σε δευτερόλεπτα. Τροποποιεί το αρχείο ~/.bash_profile για την υλοποίηση της οθόνης.
  10. Digital Footprint Finder
    Ένα εργαλείο OSINT για την ανακάλυψη δημόσιων προφίλ που σχετίζονται με ένα όνομα χρήστη σε διάφορες διαδικτυακές πλατφόρμες (κοινωνικά δίκτυα, ιστότοποι προγραμματισμού κ.λπ.). Διαθέτει μηχανισμούς προσωρινής αποθήκευσης (24 ώρες προεπιλογή), λειτουργίες stealth (λιγότερο επιθετική σάρωση) και αποθηκεύει τα αποτελέσματα σε μορφή κειμένου και JSON σε έναν φάκελο 'Digital Footprint Finder' στις Λήψεις.
  11. Smart Notes
    Μια ασφαλής εφαρμογή σημειώσεων βασισμένη σε τερματικό (TUI) που χρησιμοποιεί curses. Διαθέτει κρυπτογραφημένη αποθήκευση (στο ~/.smart_notes.json), ενσωμάτωση ημερολογίου, υπενθυμίσεις (χρησιμοποιώντας τον προγραμματιστή εργασιών του Termux μέσω ανάλυσης dateutil και προαιρετικής εκτέλεσης εντολών), υποστήριξη εμπλουτισμένου κειμένου και προηγμένες δυνατότητες αναζήτησης.
  12. QR Code Generator
    Ένα απλό εργαλείο για τη δημιουργία εικόνων κώδικα QR από κείμενο (συνήθως URLs). Αποθηκεύει την παραγόμενη εικόνα PNG σε έναν φάκελο 'QR Codes' εντός του κύριου καταλόγου Λήψεις, χρησιμοποιώντας μια απολυμασμένη έκδοση των δεδομένων εισόδου ως όνομα αρχείου.
  13. File Converter
    Ένας διαδραστικός μετατροπέας αρχείων βασισμένος σε τερματικό (TUI) που υποστηρίζει διάφορες μορφές σε έγγραφα, εικόνες (συμπεριλαμβανομένων SVG/PSD μέσω cairosvg/psd-tools), ήχο/βίντεο (απαιτεί ffmpeg), αρχεία (ZIP, RAR μέσω unrar, 7Z, TAR, GZ) και δεδομένα (CSV/JSON). Οργανώνει τα μετατρεπόμενα αρχεία σε 40 κατηγοριοποιημένους υποφακέλους εντός ενός κύριου καταλόγου 'File Converter' στις Λήψεις.
  14. Extra Content
    Ένα βοηθητικό script που αντιγράφει τον φάκελο 'Extra Content' (και όλο το περιεχόμενό του) από τον κατάλογο του έργου DedSec (`~/DedSec/Extra Content`) στον κύριο φάκελο Λήψεις (`~/storage/downloads/Extra Content`) στην αποθήκευση της συσκευής.

Νομική Αποποίηση Ευθύνης: Όλα τα εργαλεία και οι πληροφορίες που παρουσιάζονται εδώ προορίζονται αποκλειστικά για εκπαιδευτικούς και ηθικούς ερευνητικούς σκοπούς. Χρησιμοποιήστε τα αποκλειστικά σε συστήματα που σας ανήκουν ή για τα οποία έχετε ρητή, γραπτή άδεια να δοκιμάσετε. Η μη εξουσιοδοτημένη χρήση είναι παράνομη.
