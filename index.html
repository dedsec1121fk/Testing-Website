How To Install And Setup The DedSec Project

Requirements

  • Device: Android with Termux installed.
  • Storage: Min 8GB free. (Images and recordings also consume space.)
  • RAM: Min 2GB.

Step-by-Step Installation

  1. Install Termux & Add-Ons

    Note: To install APKs (e.g., F-Droid), ensure you:
    - Enable unknown sources (Settings > Security > Install Unknown Apps).
    - Download F-Droid, then get Termux from F-Droid.
    - Install add-ons: Termux:API, Termux:Styling.
    - Allow the fdroid process when prompted.

  2. Update Packages & Install Git
    Important: Open the Termux application on your device before copying and pasting the command below.
    pkg update -y && pkg upgrade -y && pkg install git nano -y && termux-setup-storage
  3. Clone the Repository
    git clone https://github.com/dedsec1121fk/DedSec
  4. Run the Setup Script
    cd DedSec && bash Setup.sh

    The script will handle the complete installation. After the process, you will see a settings menu, you must choose Change Menu Style and then choose a menu style: list or grid.
    After that close Termux from notifications and reopen it.

Tip: After setup, you can quickly open the menu by typing e (for the English version) or g (for the Greek version) in Termux.

Basic Tools

File Converter.py

An interactive, curses-based tool for Termux that converts a wide variety of files. It supports documents (PDF, DOCX, etc.), images (JPG, PNG, WEBP, etc.), audio (MP3, WAV, etc.), and video (MP4, MKV, etc.). It automatically manages dependencies and organizes all converted files into specific folders.

Save Location: All converted files are saved within the /storage/emulated/0/Download/File Converter/ directory, sorted into 40 different sub-folders (e.g., PDF, JPG, MP3).

DedSec's Network.py

An advanced, no-root network and security toolkit for Termux with a TUI (curses) interface. It includes Wi-Fi scanning, Nmap wrapping, port scanning, network discovery, and various OSINT (Open Source Intelligence) tools. It also features a comprehensive security audit logger and an SSH honeypot (SSH Defender) to detect attacks.

Save Location: Creates a main directory at ~/DedSec's Network/ to store configuration, audit logs (audit_results.db), Wi-Fi scan history (wifi_scans.db), and SSH Defender logs.

Note: The integrated OSINTDS tool saves its separate reports to /storage/emulated/0/Download/OSINTDS/.

Android App Launcher.py

An fzf-based Termux utility for managing your installed Android apps. It allows you to launch, uninstall, view app info, or extract the APK file of any application. Its main feature is the "App Perm Inspector", which scans an APK to identify dangerous permissions (like location or camera access) and detects embedded ad trackers, generating a full security report.

Save Location (APK): Extracted APKs are saved to /storage/emulated/0/Download/Extracted APK's/.

Save Location (Reports): Security analysis reports are saved to /storage/emulated/0/Download/App_Security_Reports/.

Extra Content.py

A simple script designed to copy the entire 'Extra Content' folder from within the DedSec project directory into your phone's main Downloads folder. This makes accessing supplementary materials easier.

Save Location: Copies the folder to /storage/emulated/0/Download/Extra Content/.

Settings.py

The central configuration hub for the DedSec Project. It allows you to update the project and its dependencies, change the Termux prompt style, switch between list and grid menu styles, and select the project's display language (English/Greek).

Configuration File: Modifies the Termux configuration file /data/data/com.termux/files/usr/etc/bash.bashrc to apply settings.

Fox's Connections.py

A unified script that runs both the secure 'Fox Chat' application and 'DedSec's Database' file server. It requires a single secret key for access and provides both a local network URL and an optional Cloudflare tunnel URL for internet access. Fox Chat supports text, voice messages, file sharing (up to 10GB), and video calls. DedSec's Database allows secure file uploads and management.

Database Save Location: Files uploaded to DedSec's Database are stored in /storage/emulated/0/Download/DedSec's Database/.

Chat Files: Files shared in Fox Chat are handled within the chat interface and downloaded directly by participants.

Smart Notes.py

A terminal-based (curses) note-taking application. It allows creating, editing, deleting, and searching notes. It features a reminder system that can parse due dates (e.g., 'tomorrow 5pm') and optionally run shell commands when a reminder is due.

Notes File: Stores all notes in ~/.smart_notes.json.

Configuration File: Stores settings (like preferred editor) in ~/.smart_notes_config.json.

QR Code Generator.py

A straightforward script that takes a URL as input and generates a QR code image for it. It automatically installs the necessary 'qrcode' library if missing.

Save Location: Saves the generated QR code images (as PNG files) into /storage/emulated/0/Download/QR Codes/.
