Learn About The Tools

Discover the powerful cybersecurity tools and features available in the DedSec Project toolkit.

Toolkit Overview

Basic Toolkit

Android App Launcher

A utility to manage Android apps directly from the terminal. It can launch apps, extract APK files, uninstall apps, and analyze security permissions. Advanced Android application management and security analysis tool. Features include app launching, APK extraction, permission inspection, security analysis, and tracker detection. Includes comprehensive security reporting for installed applications.

Dark

A comprehensive dark web intelligence and reconnaissance platform.

DedSec's Network Toolkit

A lightweight, non-root optimized network toolkit focused on OSINT, Web Auditing, and Connectivity. It has been refactored for stability and speed.

Username Scanner

A powerful OSINT tool designed for extensive digital footprint analysis. It rapidly scans over 250 websites for a target username, including major social media platforms, tech sites, gaming, and financial services. It handles rate-limits and automatically manages dependencies to provide a comprehensive, saved report.

Advanced Utilities

Extra Content

A Swiss Army knife for file management, conversion, and organization. This utility handles tasks from creating HTML websites using templates to complex format conversions (image, document, audio/video). It also includes features for archive extraction, folder organization, and multi-image to PDF conversion.

QR Code Generator

A simple yet effective tool for instantly generating QR codes directly within the terminal. It supports various data types (URLs, text, contact info) and automatically saves the high-quality, scannable QR code image to a dedicated downloads folder.

Settings

The core configuration and maintenance hub for managing the DedSec Project's environment and settings.

Startup Theme

Customizes your Termux startup with ASCII art loading screens. Customizable ASCII art loading screen system for Termux startup. Features automatic installation, custom art support, adjustable delay timers, and seamless integration with Termux bash configuration. Includes automatic cleanup to ensure one-time display and global bashrc patching for delayed script execution.

Load Testing

Sod

A comprehensive load testing tool for web applications, featuring multiple testing methods (HTTP, WebSocket, database simulation, file upload, mixed workload), real-time monitoring, and detailed reporting. It includes robust error handling, proxy support, and customizable test scenarios for thorough performance evaluation.

Social Engineering Tools

Site-Cloner

A phishing tool that clones a target website's login page for credential capture. It uses HTML, CSS, and JS to create a perfect replica, captures entered credentials, and provides them via cloudflare tunneling. Advanced phishing simulation tool for creating perfect login page replicas to capture credentials. Features include a fully responsive design, dual data storage for credentials and images, timestamped file organization, and integrated Cloudflare tunneling.

Simple Messaging App

A sophisticated, full-featured messaging and collaboration platform. Offers real-time messaging, WebRTC video and voice calls, large file sharing (up to 50GB), and an integrated file database. It utilizes single-secret-key authentication and Cloudflare tunneling for secure, easy access.

Fake Card Details Page

Phishing Tool. A convincing fake credit card detail form, themed as an Antivirus renewal page with a fake $3 charge. It's designed to capture full credit card details (Name, Number, Expiry, CVV) using social engineering tactics. It provides a highly realistic interface to maximize submission rates.

Camera Phishing

Phishing Tool. A social engineering page that tricks the user into allowing camera access, capturing a photo using their front camera without their knowledge. Features automatic photo capture upon permission grant, and saves the image with a timestamp to the downloads folder. It utilizes a professional, yet generic, UI theme to minimize suspicion.

Location Phishing

Phishing Tool. Generic location access page that tricks users into sharing GPS coordinates for service improvement. Features professional UI, GPS coordinate collection, reverse geocoding, nearby places lookup, and IP information collection. Saves enriched location data in JSON format.

Fake Facebook Friends Page

Facebook-themed phishing page designed to maximize credential capture. The page mimics the official Facebook login with a focus on 'Connect with friends' to increase user trust. It captures login credentials and automatically tunnels the page using Cloudflare for remote access.

Fake TikTok Followers Page

A highly realistic TikTok-themed phishing page, leveraging social desire (free followers) to capture login credentials. It features official-looking design elements, multiple login options, feature highlights, and convincing call-to-action. Uses connection psychology to lure users into entering credentials. Automatically generates public access links via cloudflare tunneling.

Important Information

All tools and information presented here are for educational and ethical research purposes only. Users are responsible for ensuring they have proper authorization before using these tools on any network or system they do not own. Social engineering tools should only be used in authorized penetration testing scenarios with explicit written permission.
