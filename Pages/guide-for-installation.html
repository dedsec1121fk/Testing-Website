Guide For Installation

Step-by-step instructions to install and setup the DedSec Project on your Android device.

Requirements

  • Device: Android with Termux installed.
  • Storage: Minimum 8GB free space.
  • RAM: Minimum 2GB.

Installation Steps

1. Install Termux & Add-Ons

Note: To install APKs (e.g., F-Droid), ensure you enable unknown sources in Settings > Security > Install Unknown Apps.

Install F-Droid

F-Droid is an alternative app store for Android that provides free and open-source software. It's the recommended way to install Termux and other security tools.

Download F-Droid

How to Install APK Files on Android:

  1. Download the F-Droid APK from the link above
  2. Open your device's Settings > Security (or Apps & Notifications)
  3. Enable 'Install unknown apps' or 'Unknown sources' for your browser or file manager
  4. Open the downloaded APK file and tap 'Install'
  5. Once installed, open F-Droid and search for and install the following essential apps:
  • Termux: Terminal emulator with packages (The main terminal)
  • Termux:API (For device API access)
  • Termux:Styling (For terminal customization)

2. Termux Setup

Open Termux and run the following commands:

pkg update && pkg upgrade -y

Grant storage permission to Termux:

termux-setup-storage

This will ask for permission. Tap 'Allow' or 'Grant'.

3. Install Git and Clone DedSec Project

Install git (required to download the project) and then clone the repository:

pkg install git -y git clone https://github.com/dedsec1121fk/DedSec-Project.git

4. Run the Installation Script

Navigate to the project folder and run the main installer script.

cd DedSec-Project bash install.sh

Note: If running the installer fails, you might need to run the following before retrying: `chmod +x install.sh`

5. Running the Project

Once the installation is complete, you can run the main tool using the command below:

DedSec

Optional Steps

Change Style (Colors & Fonts)

You can customize the terminal's appearance if you have Termux:Styling installed. Press and hold on the screen, tap 'More...', and then select 'Style' to choose a new color scheme or font.

Updating the Project

If you have already installed the DedSec Project, you can easily update it to the latest version by running the following command inside Termux:

DedSec --update

Uninstalling the Project

To completely remove the DedSec Project and all its dependencies, run the following command in Termux:

DedSec --uninstall
