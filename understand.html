⚠️ Understanding Threats: Educational Demonstrations

CRITICAL NOTICE: The following scripts are included for educational and defensive purposes ONLY. Their function is to demonstrate how common attacks work, so you can learn to recognize and protect yourself against them. They should only be used in a controlled environment, on your own devices and accounts, for self-testing.

Toolkit Summary

Fox Chat

A secure, end-to-end encrypted chat application protected by a one-time Secret Key. Features include text messaging, voice notes, file sharing (up to 10 GB), live camera capture, and peer-to-peer video calls.

DedSec's Database

A password-protected, self-hosted file storage server.

OSINTDS

A comprehensive tool for Open Source Intelligence (OSINT) gathering and web reconnaissance.

Phishing Demonstrations

Modules that demonstrate how a malicious webpage can trick a user.
