Learn About The Tools

Discover the powerful cybersecurity tools and features available in the DedSec Project toolkit.

Toolkit Overview

DedSec's Network Toolkit

Advanced Network & Security Toolkit optimized for Termux. Comprehensive suite including Wi-Fi scanning, network discovery, port scanning, OSINT gathering, vulnerability assessment, and SSH Defender honeypot system. Runs 100% without root access on low-resource devices.

Android App Launcher

Advanced Android application management and security analysis tool. Features include app launching, APK extraction, permission inspection, security analysis, and tracker detection. Includes comprehensive security reporting for installed applications.

Digital Footprint Finder

Comprehensive OSINT tool for discovering digital footprints across multiple platforms. Features stealth mode, intelligent caching, major platform prioritization, and detailed reporting. Automatically validates usernames and saves results in multiple formats.

Extra Content Manager

Utility script for managing additional content in the DedSec toolkit. Automatically copies the 'Extra Content' folder from the DedSec directory to the device's Downloads folder, making supplementary materials easily accessible to users.

Advanced File Converter

Comprehensive file conversion tool supporting 40+ formats across images, documents, archives, audio, and video. Features an interactive curses-based UI, automatic dependency installation, and organized folder structure. Includes advanced features like multi-image PDF creation and media format conversion.

Fox Chat & Database

Unified communication and file management platform combining secure real-time chat with a comprehensive file database. Features encrypted messaging, video calls, file sharing (up to 10GB), and cloudflare tunneling for external access. Includes Instagram-style UI and unified authentication system.

QR Code Generator

Simple and efficient QR code generation tool for Termux. Automatically installs required dependencies and creates QR code images from URLs or text. Saves generated codes to a dedicated 'QR Codes' folder in Downloads with automatic filename sanitization.

DedSec Settings Manager

Comprehensive system management and configuration tool for the DedSec Project. Features include project updates, package management, language selection, menu style customization, and system information. Provides both list and grid-based navigation interfaces with full Greek/English language support.

Smart Notes

Advanced note-taking application with reminder system and TUI interface. Features include external editor integration, intelligent filtering, reminder metadata parsing, and automated command execution. Supports flexible date formats, timezone handling, and both CLI and curses-based interfaces.
